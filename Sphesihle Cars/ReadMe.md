# Sphesihle Cars

An online car store webapp built with html ,css ,js,Node.js, Express, Ejs,Postgress.

## Features

- Browse cars with prices and images
- Add cars to cart
- Session-based login and shopping
- Salting and hashing for security and https protocol.
- Responsive UI with Bootstrap

## Installation

```bash
git clone https://github.com/Sphesihlesfiso/SphesihleDev.git
cd SphesihleDev/Sphesihle Cars
npm install
npm start
```
